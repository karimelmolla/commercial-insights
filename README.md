This repository contains a business and financial data analysis project covering transaction data from July to December 2024. The analysis focuses on evaluating merchant performance, city-level contribution, category profitability, promotional opportunities, and additional business insights using completed transactions only.

The project was designed to simulate a real-world business case where data-driven decisions are required to support growth, marketing strategy, and operational planning. Using tools such as Excel and data analysis techniques, the dataset was explored to identify trends, patterns, and actionable recommendations.

Key analyses included:

Identifying the Top 5 merchants by GMV (Gross Merchandise Value) for each quarter and comparing their performance trends over time.
Analyzing monthly city contribution to total GMV and highlighting how customer activity differs across locations.
Recommending merchant categories for expansion based on profitability and total GMV performance.
Selecting merchants for promotional campaigns by evaluating transaction performance, seasonal demand, and GMV ranking.
Discovering additional business insights such as seasonal spikes, high-performing cities, category trends, and merchant behavior.

The project emphasizes:

Building a data-driven scoring model (Correlation Matrix & Tier Distribution)
Data cleaning and filtering
Business KPI analysis
Profitability calculations
Trend identification
Recommendation-driven reporting
Data storytelling and decision-making

Through this analysis, the project demonstrates how transaction data can be transformed into meaningful business insights that help improve strategic planning, optimize promotions, and identify growth opportunities.
